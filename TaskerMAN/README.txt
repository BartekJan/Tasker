INSTALL INSTRUCTIONS FOR TASKERMAN AND TASKERCLI
1. Place the folder TaskerMAN which contains the web files in the desired location.
2. Access install.php in your browser before doing anything else and input database information.
3. If the information is valid you should get redirected to index.php and TaskerMAN should be ready for use.
4. For the JAVA app you need to input the database information you used previously inside the application itself.
